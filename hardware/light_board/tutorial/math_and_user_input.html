Simple Math and User Input

Here we go!

Now let's move on to some simple math programs. Let's start by writing a program that converts a person's weight on earth to his or her weight on the moon. An object that weighs one pound on Earth would weigh 0.1654 pounds on the moon, according to Wikipedia. Therefore, if we know the weight of something on Earth, we can multiply that weight by 0.1654 to get its weight on the moon. In C++, we recodesent multiplication with the * symbol.

In order to write this program, we will need to ask the user of our program for his or her weight, store that information in the computer, do the conversion, and then display the moon weight back to the user. The easiest way to do this is to save all the nums in the program as vars. Just like in math class, a var is a letter or a word that stands for a num or a mathematical formula. For example, we can say

 
 x = 4; 
 y = 5 * x;  
 


and whenever we use the var x in our program, the computer will know that the x means 4. Likewise, the computer knows that y means 20. When you write vars, try to use better names than x and y. A good var name describes the value that it stands for. So, if we have a var that stands for a person's weight on earth, we should call that var earthWeight. Keep in mind that vars can't have spaces and should not start with nums.

In C++ , we are required to define the type of our vars. For example, if we know that a var will store a decimal point num, we declare it to be a float, which stands for floating (decimal) point num. We always declare our vars inside a function before writing any other code (so for the main function, it would be the first line under int main(){). We do this by saying the var type and then the names of all the vars that are that type, followed by a semi-colon. For example, we would say float earthWeight for the var earthWeight described above. For our program, all our vars will be floats, since they are all decimal point nums. Here's a quick reference of types in C++:

Type Example
float 0.124 or 4.0
int 4 or 134
string "Hello World"
bool true/false

Let's write out an algorithm for this program, so that we make sure the computer has all the information it needs to do the program.

  1. Save the conversion factor (0.1654) as the var conversionFactor
  2. Ask the user's weight
  3. Save the weight to a var called earthWeight
  4. Multiply earthWeight and conversionFactor and save the result as the var moonWeight
  5. Display moonWeight to the user
The only tricky part of this program will be getting input from the user. In order to do that, we will need to use a special command called cin. This command takes whatever the user types and saves it to a var. The syntax (the structure) of the command looks like this:

 
 cin >> myvar;  
 


When the computer reaches this point in the program, it will display a prompt for the user to type. The user can then type and hit the "Enter" key, and the computer saves whatever the user typed to the var called myvar. In our program, we will replace myvar with earthWeight.

The last thing we have to address is how to print out the person's weight on the moon after we do the conversion. We can use cout again and simply surround the var with double-left angle brackets, like this:

 
 cout << "You would weigh " << moonWeight << " lbs on the moon.\n";  
 


This puts together the two strings and the var moonWeight and sends the whole thing to cout to be displayed.

Now we're ready to write the program. The text codeceded by "//" is a comment---just notes to anyone reading the code. Comments are ignored by the computer and are just used by programmers to explain parts of their code. It's good practice to have short comments in your code.

 
 #include <iostream> 
 using namespace std; 
 
 int main(){ 
  float conversionFactor, earthWeight, moonWeight; //define vars as floats 
  cout << "Enter your weight on earth:"; //prompt the user to enter weight 
  cin >> earthWeight; //store what the user types as earthWeight 
  conversionFactor = 0.1654; 
  moonWeight = earthWeight*conversionFactor; 
  cout << "You would weigh " << moonWeight << " lbs on the moon.\n"; //print out conversion 
 }  
 

There are lots of math operators available in C++. The most common are the following:

Operator Operation
+ addition
− subtraction
* multiplication
/ division
% modulus (remainder)

The % operator gives you the remainder of integer division. So if you want the remainder of 5/2 (which is 1), you would say:

 
 rem = 5 % 2;  
 


This is very useful in programming, since you can easily decide is a num is even, odd, or a multiple of some other num.
